# Nome do APP
Gerenciador de Podcast

### Descrição
App estilo netflix para centralizar diferentes episódios de podcasts separados por categorias

### Funcionalidades
- Listagem episódios de podcasts em seções de categorias
  - [saúde, fitness, mentalidade, humor]
- Filtrar episódios por podcast

### Como

#### Feature:
Listar episódios dos podcasts em sessões de categorias

#### Implementação
Retornar API rest: nome do podcast, nome do episódio, imagem de capa, link 

```js
{
    podcastName: "flow",
    episode: "nome do episódio"
}
```
